# Pixel Agents Tonone Asset Sources

Installed source packs:

- OpenGameArt: Sci-fi Interior tiles by Buch, CC0.
  - https://opengameart.org/content/sci-fi-interior-tiles
  - Downloaded as `downloads/scifitiles-sheet.png`.
- OpenGameArt: Warped Top-Down Tech Lab 2 by ansimuz, CC0.
  - https://opengameart.org/content/warped-top-down-tech-lab-2
  - Downloaded as `downloads/top_down_lab_2_files.zip`.
- OpenGameArt: 3 Cyberpunk Characters by CraftPix.net, OGA-BY 3.0.
  - https://opengameart.org/content/3-cyberpunk-characters
  - Downloaded as `downloads/3_cyberpunk_characters.zip`.

Pixel Agents loadable assets:

- `assets/characters/char_0.png` through `char_5.png` are Pixel Agents-format cyber variants derived from the bundled Pixel Agents character sheets so they preserve the required 7-frame, 3-direction layout.
- `assets/furniture/TONONE_*` contains Pixel Agents furniture manifests and PNG props for the cyber room layout.

The itch.io candidate pages saved under `downloads/*.html` were not directly installed because their pay-what-you-want pages require itch's download flow instead of stable public file URLs.
